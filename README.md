# Obsidian Web Clipper Templates
This repository contains my collection of Obsidian Web Clipper templates.

## List of Templates
- [Bookmarks](templates/bookmark-clipper.json) ... Simple bookmarks that can be viewed via Obsidian bases like a bookmark manager. Tags are auto-generated via Interpreter variable.
- [Clippings](templates/clipping-clipper.json) ... Web clippings (selections, highlights, whole articles) with auto-generated topics and tags.
